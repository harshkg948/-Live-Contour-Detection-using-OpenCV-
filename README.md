# 🎥 Live Contour Detection using OpenCV 🚀

A beginner-friendly Computer Vision project built using **Python 🐍** and **OpenCV 👁️** that detects and draws contours from a **live webcam feed** in real time.

---

## ✨ Features

✅ Real-time webcam capture
✅ Grayscale image conversion
✅ Binary thresholding
✅ Contour detection
✅ Live contour visualization
✅ Beginner-friendly OpenCV project

---

## 🛠️ Tech Stack

* 🐍 Python
* 👁️ OpenCV

---

## 📸 How It Works

1️⃣ Captures live webcam frames
2️⃣ Converts frames to grayscale
3️⃣ Applies binary thresholding
4️⃣ Detects contours from objects
5️⃣ Draws green contour boundaries live

---

## 📂 Project Structure

```bash
contours.py
```

---

## ▶️ Run the Project

### 📦 Install OpenCV

```bash
pip install opencv-python
```

### 🚀 Run the Script

```bash
python contours.py
```
